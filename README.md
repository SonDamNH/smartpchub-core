## 📖 Giới thiệu hệ thống

**Smart PC Hub** là nền tảng thương mại điện tử chuyên biệt cung cấp **PC Gaming cao cấp** và **linh kiện máy tính**. Hệ thống được xây dựng với mục tiêu mang lại trải nghiệm mua sắm mượt mà trên giao diện công nghệ hiện đại theo phong cách **Premium Tech UI** với độ tương phản cao và hiệu ứng Glassmorphism độc đáo.

Toàn bộ hệ thống được **container hóa bằng Docker**, đảm bảo triển khai nhất quán trên mọi môi trường, không phụ thuộc vào cấu hình máy cá nhân.

## ⚙️ Hướng dẫn cài đặt

> 💡 Hệ thống được đóng gói hoàn toàn bằng Docker. Chỉ cần **Docker** và **Git** là đủ để chạy toàn bộ dự án trên bất kỳ máy nào.

### Yêu cầu môi trường

| Công cụ | Tải về |
|:---|:---|
| Docker Desktop / Engine | [docker.com](https://www.docker.com/products/docker-desktop/) |
| Git | [git-scm.com](https://git-scm.com/) |
| Ngrok *(cho Webhook PayOS)* | [ngrok.com](https://ngrok.com/) |

### Các bước cài đặt

**① Clone mã nguồn về máy**

```bash
git clone https://github.com/SonDamNH/smartpchub-core.git
cd smartpchub-core
```

**② Khởi chạy hệ thống bằng Docker**

```bash
docker-compose up -d
```

> ✅ Docker sẽ tự động build container WordPress + MySQL và kết nối chúng.  
> 🌐 Sau khi hoàn tất, truy cập hệ thống tại: **http://localhost**

**③ *(Tùy chọn)* Kích hoạt Webhook thanh toán PayOS qua Ngrok**

Mở một terminal mới và chạy:

```bash
ngrok http 80
```

Sao chép URL `https://` mà Ngrok cấp → Vào **WP Admin → PayOS Settings → Webhook URL** và dán vào.

## ▶️ Hướng dẫn chạy project

```bash
# ✅ Khởi chạy toàn bộ hệ thống (chế độ nền)
docker-compose up -d

# 📋 Xem log realtime của các container
docker-compose logs -f

# 🔄 Khởi động lại hệ thống
docker-compose restart

# ⏹️  Dừng toàn bộ hệ thống
docker-compose down

# 🗑️  Dừng và xóa toàn bộ dữ liệu volume (RESET HOÀN TOÀN)
docker-compose down -v
```

> ⚠️ **Cảnh báo:** Lệnh `down -v` sẽ **xóa toàn bộ dữ liệu** trong database. Chỉ dùng khi muốn reset hoàn toàn hệ thống.
